# Freelio — Business Dashboard

**Freelio** is a modern **freelancer business management dashboard** designed to help freelancers manage clients, projects, invoices, analytics, and payments in a single interface.

The application focuses on **clean UI, interactive analytics, and productivity tools** for independent professionals.

---

# Features

## Dashboard Overview

The dashboard provides a quick overview of business performance including:

* Total earnings
* Client count
* Active projects
* Pending payments

Users can quickly understand their **business performance and workflow status**.

---

## Analytics & Insights

Freelio includes interactive data visualizations to track business metrics:

* Revenue growth chart
* Client distribution analytics
* Revenue vs expenses comparison
* Project status distribution
* Monthly performance insights

Charts are powered by **Chart.js** for smooth and responsive visualizations.

---

## Project Management

Freelio allows users to manage multiple projects efficiently:

* Project status tracking
* Project deadlines
* Project value monitoring
* Progress bars for project completion
* Client association with projects

---

## Client Management

The client directory enables users to:

* Manage client information
* Track number of projects per client
* Monitor client payments
* View client acquisition sources
* Track retention rates

---

## Invoice & Payment Tracking

Freelio provides financial management tools including:

* Invoice creation
* Payment tracking
* Pending invoice alerts
* Payment method analytics
* Exportable financial reports

---

## Activity Feed

The dashboard includes an activity timeline showing:

* New client onboarding
* Payment confirmations
* Project completions
* Upcoming deadlines

This helps freelancers stay updated with recent business activity.

# How It Works

1. The dashboard loads the **overview page by default**.
2. Sidebar navigation switches between different sections.
3. JavaScript dynamically updates page content and charts.
4. Analytics charts are **lazy-loaded** when the analytics page is opened.
5. Users can create new invoices, projects, and clients using modal forms.
